# Bills Dashboard.

## Installation & Running the Project

### prereq

- node.js 18+
- package manager.

### Installation Steps

1. **Clone the repo**

   ```bash
   git clone <repourl>
   ```

2. **Install deps**

   ```bash
   npm install
   ```

3. **Run the dev server**
   ```bash
   npm run dev
   ```
   you can see the application at `http://localhost:3000`

## Component Libraries

- **Radix UI**
- **Lucide React**
- **Tailwind CSS**

# data

- Bills are stored as objects with id, type, date, amount, and status for easy filtering and sorting
- **Mock Data**: Sample bill data is sourced from `lib/data.ts` for demonstration purposes

# Assumptions

- The application operates with client-side data (no API backend required for this task)
- Mobile navigation uses a sidebar that can be toggled for better space utilization
